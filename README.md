# Libgen Downloader

## Requirements 
```
Python >= 3.7
Git 
VirtualEnv
```

## Setup

- Clone the Repository

  ```
  git clone https://github.com/arbkm22/libgen-downloader.git
  cd libgen-downloader
  ```

- Setup a Virtualenv
  ```
  virtualenv .
  # start virtual env
  source bin/activate
  ```
- Installing requirements
  ```
  pip install -r requirements.txt
  ```
- Usage
  ```
  run the following:
  python main.py

  and follow on-screen commands
  ```
